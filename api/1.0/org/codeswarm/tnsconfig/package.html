package tnsconfig

This package utilizes the parsing capability of org.codeswarm.orafile to provide an API specific to the TNS names file "tnsnames.ora".

The project was designed to support a simple use case: Given a particular TNS alias, retrieve a single TnsAddress.

val address: Option[TnsAddress] = TnsConfig("your_database_alias").address

See apply methods on the TnsConfig object for more options, and the TnsConfig class for more detail on all of the output information it contains.
